# Minimal Neovim Config

A clean, minimal setup for Neovim with:

- Telescope (file search)
- Render Markdown (live markdown rendering)
- Transparent background
- Markdown checkbox toggle
- Leader key mappings

```bash
mkdir -p ~/.local/share/nvim/site/pack/plugins/start

git clone https://github.com/nvim-lua/plenary.nvim \
~/.local/share/nvim/site/pack/plugins/start/plenary.nvim

git clone https://github.com/nvim-telescope/telescope.nvim \
~/.local/share/nvim/site/pack/plugins/start/telescope.nvim

git clone https://github.com/MeanderingProgrammer/render-markdown.nvim \
~/.local/share/nvim/site/pack/plugins/start/render-markdown.nvim
```
