# KDE Plasma Configuration Files

Location (user-specific configs):

~/.config/

---

## 1. kcminputrc

Path:
~/.config/kcminputrc

Controls:

- Mouse speed
- Mouse acceleration
- Touchpad settings
- Keyboard repeat rate
- Input device behavior

---

## 2. kdeglobals

Path:
~/.config/kdeglobals

Controls:

- Global color scheme
- Fonts
- Icon theme
- Widget style
- General UI appearance settings

---

## 3. kglobalshortcutsrc

Path:
~/.config/kglobalshortcutsrc

Controls:

- Global keyboard shortcuts
- Custom keybindings
- System-wide hotkeys

---

## 4. krunnerrc

Path:
~/.config/krunnerrc

Controls:

- KRunner settings
- Enabled search plugins
- Search behavior and appearance

---

## 5. kscreenlockerrc

Path:
~/.config/kscreenlockerrc

Controls:

- Lock screen appearance
- Lock timeout
- Lock screen behavior

(Note: This is NOT SDDM login screen config.)

---

## 6. kwinrc

Path:
~/.config/kwinrc

Controls:

- Window manager behavior
- Focus policy
- Window effects and animations
- Tiling settings
- Border behavior
- Compositor settings

---

## 7. kwinrulesrc

Path:
~/.config/kwinrulesrc

Controls:

- Custom window rules
- Forced window size/position
- App-specific window behavior

---

## 8. plasma-org.kde.plasma.desktop-appletsrc

Path:
~/.config/plasma-org.kde.plasma.desktop-appletsrc

Controls:

- Panel layout
- Taskbar configuration
- Desktop widgets
- Panel position and size
- Widget configuration

(Resetting this resets your panel layout.)

---

## 9. plasmarc

Path:
~/.config/plasmarc

Controls:

- General Plasma behavior
- Panel visibility behavior
- Some startup/session preferences
